# Santa Expression Images

Place your 5 Santa expression images in this folder:

1. **santa-grumpy.png** - For jolliness 0-20
   - Grumpy, tired, or frustrated expression

2. **santa-neutral.png** - For jolliness 21-40
   - Neutral, calm expression

3. **santa-slight-smile.png** - For jolliness 41-60
   - Small smile, warming up

4. **santa-happy.png** - For jolliness 61-80
   - Happy, cheerful expression

5. **santa-jolly.png** - For jolliness 81-100
   - Maximum jolliness, big smile, very happy

## Image Specifications

- Format: PNG (recommended for transparency) or JPG
- Size: 500x500px recommended (or similar square aspect ratio)
- Background: Transparent PNG works best
- Style: Should all be the same character with different expressions
- Character: Science Santa (Santa with teacher vibes)

## Creating the Images

You can:
- Commission an artist
- Use AI image generation (DALL-E, Midjourney, etc.)
- Create them yourself
- Use existing Christmas-themed character assets

Make sure you have the rights to use any images you add here!
